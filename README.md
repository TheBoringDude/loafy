# loafy

Fullstack blogging web app build with Nextjs, Prisma and PlanetScale DB.

## Features

- [x] Full-features markdown editor for creating posts.
- [x] Show all posts and view each.
- [x] Single-user blogging platform.
- [x] Paginated posts.
